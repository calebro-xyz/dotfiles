# Dotfiles

This repository contains all my personal dotfiles for configuring various development tools, shell environments, and editors.

**Work in Progress (WIP)**: This collection is still being organized and updated as I refine my setup.

## Overview

These dotfiles include configurations for:

- Shell (Bash, Zsh)
- Text editors (Vim, Neovim, VS Code)
- Terminal tools (Tmux, Git)
- And more...

## Installation

To use these dotfiles on your system:

1. Clone the repository:

   ```bash
   git clone https://github.com/calebro-xyz/dotfiles.git ~/.dotfiles
   ```

2. Run the installation script (if available) or manually symlink the files to your home directory.

## Usage

Feel free to explore and adapt these configurations to your needs. Contributions and suggestions are welcome!
